# Using WPAD (Web Proxy Auto-Discovery Protocol) on Windows

The **WPAD** (Web Proxy Auto-Discovery) protocol allows you to easily configure the proxy settings on the clients in your network. The idea behind WPAD is that a client can use DHCP and/or DNS to find a web server on your network that has a PAC configuration file with proxy setting
