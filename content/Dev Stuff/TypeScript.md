Pretty much like JavaScript but we some bonus features
It allow us to run Javascript outside the browser.
## Check version 
```
tsc -v
```
## Installing Windows
It runs on node JS so we need to install node
## RUN
```
node .\script.ts
```

#### Troubleshooting

```
npm i -D tsx
npx tsx src/index.ts
```
